# UI/UX Pro Max - Usage Guide

## Quick Start

The UI/UX Pro Max skill is now installed in your Kiro workspace. You can use it by referencing the steering file in your prompts.

## How to Use

### In Kiro Chat

When you need UI/UX design help, you can:

1. **Direct Request:**
   ```
   Build a landing page for my SaaS product
   ```

2. **With Skill Reference:**
   ```
   /ui-ux-pro-max Build a landing page for my SaaS product
   ```

3. **For Design Review:**
   ```
   Review this UI component for design best practices
   ```

## How It Works

1. **You describe** what you want to build
2. **Skill analyzes** your requirements
3. **Skill searches** the design database for:
   - Matching UI styles
   - Appropriate color palettes
   - Suitable font pairings
   - Relevant UX guidelines
4. **Skill recommends** the best design system
5. **Skill generates** code with:
   - Proper colors
   - Correct fonts
   - Spacing & layout
   - Best practices

## Tips for Best Results

1. **Be specific** about your product type
2. **Mention your tech stack** (or use default HTML + Tailwind)
3. **Describe the style** you want (modern, minimal, playful, etc.)
4. **Include industry** if relevant (SaaS, healthcare, fintech, etc.)
5. **Specify features** you need (dark mode, animations, etc.)
