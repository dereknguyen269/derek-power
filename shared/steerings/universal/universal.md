# Universal Agents

Cross-stack agents that adapt to any technology. Use when no framework-specific specialist is available.

## Available Agents

| Agent | File | Purpose | When to Use |
|-------|------|---------|-------------|
| **API Architect** | `universal/api-architect.md` | API design & specifications | New/revised API contracts |
| **API Security Audit** | `universal/api-security-audit.md` | REST API security audits | Auth reviews, compliance validation |
| **Backend Developer** | `universal/backend-developer.md` | Polyglot backend implementation | Server-side code, any language |
| **Frontend Developer** | `universal/frontend-developer.md` | Universal UI builder | User-facing code, any framework |
| **Full Stack Developer** | `universal/full-stack-developer.md` | End-to-end web apps | Complete application development |
| **Next.js Pro** | `universal/nextjs-pro.md` | Next.js applications | SSR/SSG, App Router, performance |
| **Diagram Creator** | `universal/diagram-creator.md` | AWS-style diagrams | Architecture visualization |
| **Tailwind CSS Expert** | `universal/tailwind-css-expert.md` | Utility-first styling | Tailwind CSS work |

## Agent Details

### API Architect
**File**: `#[[file:universal/api-architect.md]]`

Senior API designer delivering authoritative specifications that any team can implement.

**Use PROACTIVELY for:**
- New API contracts
- API redesigns
- OpenAPI/GraphQL specifications

**Output**: `openapi.yaml` or `schema.graphql` + API Design Report

---

### API Security Audit
**File**: `#[[file:universal/api-security-audit.md]]`

Security audit specialist for REST APIs following OWASP API Security Top 10.

**Use PROACTIVELY for:**
- Authentication reviews
- Authorization checks
- Security compliance validation
- Pre-release security audits

**Output**: Security vulnerability report with risk assessment and remediation recommendations.

---

### Backend Developer
**File**: `#[[file:universal/backend-developer.md]]`

Polyglot implementer for secure, performant backend functionality.

**Languages**: JavaScript/TypeScript, Python, Ruby, PHP, Java, C#, Rust, Go

**Use when:**
- No framework-specific agent exists
- Cross-language backend work needed
- General server-side implementation

**Output**: Implementation Report with stack detection, files modified, and test results.

---

### Frontend Developer
**File**: `#[[file:universal/frontend-developer.md]]`

Universal UI builder for responsive, accessible interfaces.

**Frameworks**: React, Vue, Angular, Svelte, Web Components, Vanilla JS/TS

**Use when:**
- No framework-specific agent exists
- Cross-framework frontend work needed
- General UI implementation

**Output**: Frontend Implementation Report with accessibility scores and component list.

---

### Diagram Creator
**File**: `#[[file:universal/diagram-creator.md]]`

AWS-style visual communication for architecture and system flows.

**Diagram Types**:
- Workflow / Process flows
- High-Level Architecture
- C4 Model (Context/Container/Component)
- Sequence diagrams
- ERD / Database schemas
- Deployment / Infrastructure
- State machines

**Output**: Mermaid diagrams with AWS color palette and styling.

---

### Tailwind CSS Expert
**File**: `#[[file:universal/tailwind-css-expert.md]]`

Utility-first UI specialist for Tailwind CSS v4+.

**Use for:**
- Tailwind CSS styling
- Utility-first refactors
- Responsive component work

**Output**: Component Delivery with files, preview, and quality checklist.

---

### Full Stack Developer
**File**: `#[[file:universal/full-stack-developer.md]]`

Versatile developer for end-to-end web application development.

**Use for:**
- Complete web applications
- Frontend + backend integration
- Full-stack architecture

**Output**: Application architecture, clean code, documentation, and tests.

---

### Next.js Pro
**File**: `#[[file:universal/nextjs-pro.md]]`

Expert Next.js developer for high-performance, SEO-friendly applications.

**Use for:**
- Next.js App Router projects

- Full-stack Next.js

**Output**: Next.js implementation with TypeScript, tests, and performance insights.

## Selection Priority

```
1. Framework-specific agent (e.g., django-backend-expert)
2. Universal agent (e.g., backend-developer)
3. Core agent (e.g., code-reviewer)
```
